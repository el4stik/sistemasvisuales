GPU SHARED MEMORY (Zero Latency Transfer)

¿Cómo funciona?

A diferencia de un cable físico, Spout comparte la textura directamente en la memoria de la tarjeta de video (GPU). Esto permite que dos programas usen la misma imagen simultáneamente sin carga extra para el procesador.

Principales Usos

TouchDesigner OBS Studio Processing Unreal Engine Generative AI Web Browsers

OBS a Resolume

Usa el plugin "OBS-Spout2-Source" para enviar tu stream o webcam procesada directamente como un clip a Resolume.

IA Híbrida

Genera visuals con Stable Diffusion o ComfyUI y envíalos vía Spout para mezclarlos con tus loops locales.

Nota para Mac:

En macOS, el protocolo equivalente se llama Syphon. Funciona exactamente igual y es compatible con el mismo flujo de trabajo.
