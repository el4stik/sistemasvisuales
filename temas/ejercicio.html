EJERCICIO PRÁCTICO

Construyendo tu primer live set visual

Objetivo general

Comprender el flujo completo del VJing:

Fuente → Procesamiento → Mezcla → Salida

A través de un ejercicio práctico de exploración visual en tiempo real.

Material necesario

  • Computador con Resolume
  • Cámara web o cámara externa
  • 3 clips de video
  • 3 imágenes fijas
  • Audífonos

Parte 1 — Preparar fuentes

  • Crear 3 capas
  • Cargar clips en la capa 1
  • Cargar imágenes en la capa 2
  • Cargar cámara en la capa 3
  • Probar cada fuente individualmente

Parte 2 — Manipulación visual

  • Agregar mínimo 2 efectos por capa
  • Explorar color, forma y movimiento
  • Jugar con escalas y rotación
  • Activar feedback si el equipo lo permite

Parte 3 — Mezcla en vivo

  • Mezclar capas manualmente
  • Probar modos de mezcla
  • Construir transiciones suaves
  • Crear clímax visual

Parte 4 — Control

  • Asignar 3 controles con teclado o MIDI
  • Opacidad
  • Un parámetro de efecto
  • Cambio de clips

Mini performance

Crear una sesión visual de 3 a 5 minutos, explorando ritmo, narrativa y atmósferas.

Reto creativo

Crear una composición usando únicamente:

  • Cámara
  • Efectos
  • Feedback

Sin utilizar clips pregrabados.

Entregable

  • Captura del set armado
  • Video 30–60 segundos
  • Breve reflexión escrita

Mensaje clave

No busques clips perfectos.

Construye sistemas vivos.
