---
description: Cómo subir cambios a internet (GitHub Pages)
---

Cada vez que hagas cambios en tu código y quieras que se vean en la web:

1. **Abre la terminal** en la carpeta de tu proyecto.
2. **Prepara tus cambios:**
   ```powershell
   git add .
   ```
3. **Guarda el progreso (con un mensaje corto):**
   ```powershell
   git commit -m "Describe brevemente qué cambiaste"
   ```
4. **Súbelo a la web:**
   ```powershell
   git push
   ```

// turbo
5. **Verifica:** Entra a [https://el4stik.github.io/sistemasvisuales/](https://el4stik.github.io/sistemasvisuales/) (tarda ~1 minuto en actualizar).
